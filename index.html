Types of Operating Systems

There are different types of operating systems. Many modern operating systems depend on a graphical user interface. However, some operating systems can be run without a GUI. Some people find that using a command shell is much more empowering. For the rest of the world, however, the GUI makes it easier to communicate with the system. That is why the most popular operating systems include Windows, Mac OS, and Linux. All three have different GUI's and feel different.
